Core Features

Intelligence from cloud to edge through low-cost embedded nodes and swarm networking.

🤖

LLM Chat Engine

Interactive conversation with Claude API over HTTP. Chat-first shell where direct input goes to AI, with /commands for system control.

🔧

Tool Use

LLM-driven hardware control via function calling. GPIO, system info, LCD drawing — the AI agent orchestrates hardware capabilities dynamically.

🎨

LCD Graphics

320x240 RGB565 framebuffer with text, shapes, and drawing primitives. AI agent can draw on screen via tool calls.

🛠

OSAL Abstraction

Write once, run on FreeRTOS and RT-Thread with zero code changes. Clean claw_os.h interface hides all RTOS specifics.

🪧

Swarm Intelligence

Node discovery, heartbeat, and distributed task scheduling. Networked nodes collaborate and execute tasks in real time.

🔌

Gateway Router

Thread-safe message routing between services. Decoupled architecture enables modular, extensible service communication.

🧠

Conversation & Skill Memory

Short-term RAM ring buffer + long-term NVS Flash persistent storage. Nodes learn and recall frequently used operation patterns.

⏰

Scheduled Tasks

Timer-driven task execution and periodic automation. Set up recurring operations that run autonomously on the device.

💬

IM Integrations

Connect to Feishu, DingTalk, QQ, and Telegram as message channels. Control your hardware from any chat platform.

Architecture

Clean layered design with OSAL bridging multiple RTOS platforms.

Supported Platforms

Run on real hardware or emulate with QEMU — your choice.

ESP32-C3

RISC-V FreeRTOS ESP-IDF WiFi BLE
  • 160 MHz single-core, 400 KB SRAM
  • CMake build system via idf.py
  • Networking + AI working on QEMU
  • Flash to real hardware or emulate

QEMU vexpress-a9

ARM Cortex-A9 RT-Thread SCons Ethernet
  • RT-Thread v5.3.0 BSP
  • SCons build system
  • Boot verified on QEMU
  • Great for development and testing
